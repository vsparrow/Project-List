# Recruiter Review
Flatiron Module 5 project

A site to review job recruiters. 

Users are able to 
  * Create accounts / Login 
  
  * View / Search / Create recruiters instances

  * View / Create comments

## Tech Stack
  * React
  
  * Ruby On Rails
  
  * Bootstrap
