# Online Poker
Flatiron Module 4 pair programming project

Play poker online, make bets against your oppenent. 

Users are able to 
  * Start a new game  
  
  * View game lobby and join existing game

  * Make bets and play against opponent

#### Tech Stack
  * React
  
  * Ruby On Rails
  
  * Bootstrap
  
#### Link to Project's github folder
